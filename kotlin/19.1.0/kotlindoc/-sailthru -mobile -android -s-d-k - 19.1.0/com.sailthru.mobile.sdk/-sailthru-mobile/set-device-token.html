setDeviceToken

fun setDeviceToken(token: String?)

Pass the FCM token into the Sailthru Mobile SDK. NOTE: This method should only be called if the app implements its own FirebaseMessagingService instance, in which case it must be called in order to provide the FCM token to the Sailthru Mobile Platform. Otherwise this will be handled automatically by the Sailthru Mobile SDK.

Parameters

token

FCM token provided by FirebaseMessagingService
