trackPageviewResult

suspend fun SailthruMobile.trackPageviewResult(url: URI, tags: List<String?>?): Result<Unit>

Asynchronously registers that the given "page" has been viewed with Sailthru SPM. Note that a Content View here directly corresponds to a page view in SPM.

Return

Result representing the outcome of the request

Parameters

url

The URL of the page we're tracking a view of. Must be a valid URL with protocol (eg. http:// or https://) - this generally should correspond to the web link of the content being tracked, and the stored URL in the Sailthru content collection. This must not be null.

tags

Tags for this page. May be null.
