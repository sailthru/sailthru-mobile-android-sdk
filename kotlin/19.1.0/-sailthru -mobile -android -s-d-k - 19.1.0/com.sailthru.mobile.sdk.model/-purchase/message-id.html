messageId

var messageId: String?

Set the message ID associated with the purchase. This is required to have revenue data matched to email sends in Sailthru. Pass the identifying message_id of the email the user is coming from; this will be the value stored in the sailthru_bid cookie for your domain.Note: In some client libraries message_id should be passed as a string. The message attribution will be displayed in your Campaign Summary, Transactional Report, Purchase Log, and in Lifecycle Optimizer Metrics.
